Java Full Stack Development RoadMap

HTML

HTML is an Basic Web develpment Scripting language where you will learn to create website and give the content to the website.

CSS

CSS is an Web develpment Scripting language where you will learn to Decorate or Design the website.

Bootstrap

It is an Web develpment Template which has in-build designs using that we can create responsive website.

JavaScript

It is used to create a interactive website where we can do some interactive task like popups, dialogue box, time & date.

Core Java

Core Java covers fundamentals of Java like Operators, Decision Construct, String, Array. Also Oops, Exception and File handling concepts.

Advance Java

This will cover the advanced topics like Multithreading, Collections, IO Stream, Networking, Swing and JDBC which is considered as a JavaEE concepts.

Spring Boot

This is an Backend Framework used to write a code in easy way to connect backend with frontend.

SQL

Here we'll cover the database concept that is query language to store and manipulate the data into the database.
